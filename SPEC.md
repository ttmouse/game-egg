# 孵蛋游戏 · 像素版 — 设计规格

## 1. 项目概述

- **类型**：休闲养成游戏（单 HTML 文件，可直接浏览器打开）
- **核心循环**：管理一颗蛋的温度，喂食、互动、等待孵化
- **目标用户**：喜欢放置类、养成类游戏的玩家
- **核心约束**：无广告、像素风格、昼夜循环（白天10分钟 + 晚上5分钟）

---

## 2. 视觉与渲染规格

### 场景设置
- **视角**：俯视 / 正侧 2D 像素风格
- **分辨率**：160×144（Game Boy 复古分辨率），CSS 放大 4 倍显示（640×576）
- **像素风格**：禁用抗锯齿，所有图形 hard pixel
- **画布**：HTML5 Canvas，imageSmoothingEnabled = false

### 调色板（9色像素风）
| 用途 | 颜色 |
|------|------|
| 背景-昼 | #87CEEB（天空蓝） |
| 背景-夜 | #1a1a2e（深夜蓝） |
| 地面/桌子 | #8B4513（棕色） |
| 蛋主体 | #FFF8DC（米白）带 #D4A574（淡棕）阴影 |
| 孵蛋器 | #8B0000（深红）+ #CD5C5C（浅红） |
| 火焰/加热 | #FF4500（橙红）、#FFD700（金黄） |
| UI文字 | #FFFFFF（白）、#000000（黑） |
| 亲密度条 | #FF69B4（粉色） |
| 温度条 | #FF4500（红）→ #00BFFF（蓝）渐变 |

### 蛋的视觉阶段（5阶段）
| 阶段 | 孵化进度 | 视觉变化 |
|------|----------|----------|
| 阶段1 | 0-20% | 完整蛋，轻微晃动 |
| 阶段2 | 20-40% | 出现细小裂纹 |
| 阶段3 | 40-60% | 裂纹扩大，蛋轻微变色 |
| 阶段4 | 60-80% | 大面积裂纹，能看到里面影子 |
| 阶段5 | 80-100% | 蛋壳破碎，孵化倒计时 |

### 孵蛋器视觉
- 圆形容器，中间有蛋
- 底部有火焰像素动画（帧动画，3帧循环）
- 加热开启/关闭有明确视觉变化

### 白天/晚上天空背景
- **白天**：渐变天空 + 像素太阳 + 漂浮白云
- **晚上**：深蓝夜空 + 像素星星闪烁 + 月亮

## 3. 游戏机制规格

### 时间系统
| 参数 | 数值 |
|------|------|
| 白天时长 | 10 分钟（600秒） |
| 晚上时长 | 5 分钟（300秒） |
| 总昼夜周期 | 15 分钟 |
| 温度自然变化 | 白天自然降温慢，晚上降温快 |

### 温度机制（核心决策点）
- **最佳温度区间**：36.5°C - 38.5°C
- **温度范围**：20°C - 45°C
- **低于35°C**：孵化进度暂停
- **低于30°C**：蛋进入"受冻"状态，每天损失亲密度
- **高于40°C**：蛋进入"过热"状态，孵化进度-5%/min
- **加热器**：开启时每秒+0.3°C，消耗"电量"资源

### 电量资源
- 电量上限：100
- 加热器消耗：2点/秒（开启时）
- 每10分钟白天自然恢复：+15电量
- 每5分钟晚上自然恢复：+5电量
- 电量耗尽时：加热器自动关闭

### 喂养机制
- 食物道具：虫子（普通）、水果（中等）、神秘饲料（稀有）
- 每2小时可喂一次（游戏内时间）
- 喂养效果：+经验+亲密度
- 食物通过随机事件或成就获取

### 亲密度系统
- 范围：0-100
- 亲密度影响孵化速度（每10点亲密度 +5% 孵化速度）
- 亲密度100时解锁"特殊孵化动画"
- 蛋受冻/过热时亲密度每天-5

### 孵化进度
- 孵化进度：0-100%
- 在最佳温度区间内：+0.5%/秒
- 低于最佳温度：+0.1%/秒
- 高于最佳温度：-0.3%/秒
- 达到100%时触发孵化动画（5秒），然后宠物出生

### 宠物性格（继承机制）
宠物出生后显示性格，基于孵化过程中的行为：
| 行为 | 性格标签 |
|------|----------|
| 经常抚摸互动 | 亲人、活泼 |
| 经常喂食 | 贪吃、圆润 |
| 温度一直稳定 | 冷静、耐心 |
| 经常冷/过热 | 坚强、倔强 |
| 忽视（少互动） | 胆小或调皮 |

## 4. 交互规格

### 像素按钮布局（底部）
- **[🔥 加热]**：开关孵蛋器
- **[🫂 抚摸]**：增加亲密度，有5秒冷却
- **[🍖 喂食]**：打开食物选择弹窗
- **[📖 图鉴]**：查看已孵化宠物

### 状态显示（顶部 HUD）
- 孵化进度条
- 温度计（数字 + 像素条）
- 电量条
- 亲密度心形图标
- 昼夜指示图标（☀️/🌙）+ 剩余时间
- 当前时间（游戏内）

---

## 5. 随机事件系统

### 事件类型
| 类型 | 事件 | 效果 |
|------|------|------|
| 昼-好 | 晴空万里 | 孵化进度+5% |
| 昼-好 | 小虫靠近 | 获得虫子×1 |
| 昼-坏 | 气温骤升 | 温度+3°C（可能过热） |
| 昼-坏 | 蚊虫骚扰 | 亲密度-5 |
| 夜-好 | 月色温柔 | 亲密度+10 |
| 夜-好 | 流星划过 | 孵化进度+8% |
| 夜-坏 | 寒流来袭 | 温度-5°C |
| 夜-坏 | 做噩梦 | 亲密度-5 |

- 白天每2分钟触发一次随机事件
- 晚上每1.5分钟触发一次随机事件
- 事件显示为像素对话框气泡，持续5秒

---

## 6. 音效（可选，静音默认）

- CSS/JS 生成 8-bit 风格音效（可选开关）
- 孵化成功：胜利音效
- 加热开启/关闭：点击音效
- 抚摸：轻快音效
- 随机事件：提示音

---

## 7. 技术实现

- **单 HTML 文件**：所有 CSS/JS 内联
- **Canvas 渲染**：160×144，低分辨率放大显示
- **无外部依赖**：不加载任何 CDN 资源
- **无广告**：纯本地游戏
- **存档**：localStorage 自动保存（每30秒）

---

## 8. 开发优先级

1. **Phase 1**：基础渲染（像素蛋、天空背景、昼夜循环）
2. **Phase 2**：温度系统 + 加热器
3. **Phase 3**：孵化进度 + 蛋视觉变化
4. **Phase 4**：亲密度 + 抚摸 + 喂养
5. **Phase 5**：随机事件系统
6. **Phase 6**：宠物性格 + 孵化动画 + 图鉴
