## 宠物战斗系统（新增，重大功能）

### 入口
- 选中2-4只宠物后，gameSelect 界面出现 ⚔️ 宠物战斗按钮

### 战斗机制
- 回合制：玩家在 Canvas 上点击技能按钮，再点击目标宠物头像
- 每只宠物3个技能：撞击 + 元素技 + 治愈
- HP = 星级 × 25；败者休息1小时（restUntil = now + 3600s）
- 伤害计算：基础威力 × (1+level×0.2) × (1+(star-1)×0.3) × 基因相克倍率
- 基因相克：火>草>水>火，光>暗>光（×2），逆×0.5

## 场景切换按钮修复

### 问题
左右箭头按钮（◀/▶）无法点击切换场景，控制台报错 `doSceneSwitch is not defined`

### 根因
`doSceneSwitch` 函数被定义在 canvas click 事件回调内部（原 line 702），不在全局作用域，导致 HTML onclick 属性和 document 级点击监听器无法访问

### 修复
1. **将 `doSceneSwitch` 移到全局作用域**：从 canvas click handler 内部移到 line 598（`getTouchPoint` 函数之后，canvas 监听器之前）
2. **画布点击区域统一调用 `doSceneSwitch`**：原来直接创建 sceneTransition 对象，改为调用统一的 `doSceneSwitch` 函数
3. **按钮 onclick 改为动态目标**：`doSceneSwitch(g.currentScene==='pets'?'incubator':'pets')`，自动根据当前场景切换到对方场景
4. **文档级点击捕获同步修复**：同样改为动态目标

### 关键代码位置
- `doSceneSwitch` 函数：line 598（全局作用域）
