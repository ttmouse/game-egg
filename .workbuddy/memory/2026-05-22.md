## 互动按钮系统（新增）

### 功能
点击底部「🎮 互动」按钮 → 选择宠物 → 选择游戏 → 进入游戏画面

### 新增内容
- HTML 按钮：`#btnInteract`（id: btnInteract, onclick: showInteractSelect()）
- 状态变量：`g.interactMode`, `g.selectedInteractPet`, `g.interactGame`
- 函数：
  - `showInteractSelect()` - 显示可互动宠物列表
  - `selectInteractPet(petId)` - 选中后跳转游戏选择
  - `showGameSelect()` - 显示游戏选择（扔球）
  - `startBallGame()` - 开始扔球游戏
  - `updateBallGame(dt)` - 每帧更新游戏状态
  - `drawBallGame(ctx)` - 渲染游戏画面
  - `endBallGame()` - 结算奖励
  - `cancelInteract()` - 取消互动
- 扔球游戏机制：
  - 拖拽释放抛球（方向+力度）
  - 球有抛物线轨迹（重力）
  - 宠物自动追逐落点
  - 85% 接住概率
  - 5 次扔球机会，结算亲密+经验
