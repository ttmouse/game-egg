## 2026-05-20 场景切换黑屏修复

### 问题
点击左右箭头切换场景后画面变黑。

### 根因
主循环中存在两套渲染逻辑冲突：
1. 背景绘制（drawBg/drawSunMoon）直接画到主 ctx
2. 过渡动画创建离屏 canvas 绘制场景后用 drawImage 叠加

过渡期间：
- 旧场景离屏 canvas 没有包含完整内容（背景+场景元素）
- 新场景偏移量计算可能有问题导致叠加异常

### 修复方案
过渡期间使用**双离屏 canvas**方案：
1. 旧场景完整渲染到 oldCanvas（背景+太阳月亮+场景内容）
2. 新场景完整渲染到 newCanvas（背景+太阳月亮+场景内容）
3. 根据 ease 进度计算两个 canvas 的 x 偏移量
4. 直接用 ctx.drawImage(oldCanvas, oldOffset, 0) 和 ctx.drawImage(newCanvas, newOffset, 0)

关键改动：
- drawBgToCtx 和 drawSunMoonToCtx 都要在离屏 canvas 中调用
- 去掉 sceneTransition.active 时对 drawBg/drawSunMoon 的重复调用（已被包含在离屏 canvas 中）
